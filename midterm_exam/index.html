Welcome to my page

There are many ways to learn how to create a web document. As you make this exciting journey, remember these important aspects that will help you to become a better developer:

  1. Avoid using spaces in your file names
  2. Use lowercase type in your element tags
  3. Backup your work frequently
  4. Make sure that your markup validates
  5. Have fun!

For more information on how to create valid web pages, visit the W3c.org
